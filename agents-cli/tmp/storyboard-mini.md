## Style Plan
- 视觉基调：写实电影感赛博朋克；雨夜高反差湿反光；颗粒感适中，轻微镜头眩光与雨滴前景。
- 色彩策略：主色霓虹青/蓝 + 辅色品红/琥珀；阴影偏冷，人物肤色压低饱和；高光来自招牌与车灯反射。
- 镜头语言：动感手持 + 稳定器混用；广角夸张速度感（24–28mm 等效）+ 少量中焦压迫（50mm）；快切与声桥推动节奏；雨声贯穿。
- 角色一致性锚点：
  - 追逐者A（被追者）：短发，透明雨披内穿浅色机能夹克；左脸颊有细小发光贴片；随身一只小型数据匣（腰侧磁扣）。
  - 追逐者B（追赶者）：黑色长风衣，肩部有硬质护甲轮廓；单眼红色HUD反光；手持折叠式电击棍（非致命）。
- 负面约束：不出现血腥肢解/过度暴力细节；不出现裸露/性暗示；不出现未成年人；不出现警徽/真实机构标识；避免“赛博”元素漂移到蒸汽朋克。

## Storyboard Script
1) 镜头1（4s）
- 画面：极近景脚步与积水，A的靴子踏出水花，霓虹在水面碎裂成青蓝色条纹；前景雨滴挂在镜头边缘。
- 镜头运动：低机位跟拍 + 轻微抖动（手持感），速度逐步加快。
- 台词/字幕：字幕（屏幕下方小字）：“别回头。”
- 音效：雨声密集；急促呼吸；远处磁悬车掠过的电机啸声。
- 转场：声桥进入下一镜（脚步声不断）。

2) 镜头2（4s）
- 画面：广角巷道，霓虹招牌与蒸汽排风口；A冲入画面右侧，B从后方暗处追出，红HUD反光一闪；路面反光像镜子。
- 镜头运动：稳定器横移追随A，最后小幅鞭甩（whip pan）带出B的出现。
- 台词/字幕：无（只留喘息与雨噪）。
- 音效：金属踏步回响；电击棍展开“咔哒”一声；远处警报样的广告音（非警笛）。
- 转场：鞭甩带动动切。

3) 镜头3（4s）
- 画面：中景，A跃上狭窄高架人行桥，桥下是车流光轨；A手摸腰侧数据匣确认仍在；B在桥头抬手甩出电击棍，电弧在雨中短促跳动（保持安全、非血腥）。
- 镜头运动：仰角推近A到半身，随即快速拉焦到后方B的红HUD反光；雨丝形成速度线。
- 台词/字幕：A低声（气音）：“上传……现在。”
- 音效：电弧噼啪；风声变大；桥体共振的低频。
- 转场：匹配剪辑到闪烁光源（招牌/反射）。

4) 镜头4（4s）
- 画面：近景在玻璃橱窗反射中完成“反转”：A贴近门禁面板，发光贴片与面板同频闪烁，门缝冷光打开；A侧身滑入，门将合未合时，B的风衣剪影逼近停在门外，红HUD反光被门缝切成一条线。
- 镜头运动：固定镜头（更紧张），仅用人物进入/门缝变化制造节奏；最后0.5s轻微推近门缝光线。
- 台词/字幕：字幕：“链接完成 98%”
- 音效：门禁蜂鸣；雨声被门内隔绝变闷；B的脚步在门外停住的刹声。
- 转场：硬切黑（或门缝合上即黑）。
